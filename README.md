# Image-Processing-repo
## description
    - serving images with size
## end points

### /api/image

-   required params
    -   filename
-   optional params
    -   width , height
-   response
    -   image file with sent paramter size

### /\*

-   response
    -   route not exist

## run scripts

- npm start
    - for development mode

- npm run start-build
    - for run in production mode

- npm run test
    - for run testing enviroment

- npm run build
    - for dist folder
